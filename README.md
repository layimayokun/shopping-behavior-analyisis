# shopping-behavior-analyisis
This is a complete analysis of the customers behavior and sales performance, the aim to uncover insights an to improve general sales and customer satisfaction
This dashboard analyzes customer purchasing behavior and revenue trends to support business decision-making. It compares historical and current performance, explores buying patterns, and evaluates customer satisfaction using both transactional and rating data.
Questions Answered
What was the previous total revenue vs current revenue?
How much clothing was purchased by season?
How does revenue differ between customers who used promo codes and those who didn’t?
Do top buyers (≥ $70) tend to subscribe or not?
What payment methods do customers prefer?
Who are the top buyers by gender?
What product size is purchased most frequently?
Analysis Features
Interactive category slicer for dynamic exploration
Custom customer satisfaction logic based on:
Change in purchase amount vs previous period
Customer rating threshold (≥ 2.5)
Satisfaction outcomes classified as Satisfied, Not Satisfied, or Indifferent
Multiple DAX measures used to calculate revenue, segmentation, and behavioral indicators
Deliverables
Interactive Power BI dashboard
Clean data model with calculated columns and measures
.pbix file included for review and exploration
Purpose
This project demonstrates my ability to translate customer data into structured insights using Power BI, combining revenue analysis, segmentation, and behavioral logic.
